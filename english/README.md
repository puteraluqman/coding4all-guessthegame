# Guess the Game

A Python guessing game built for the **Coding for All – Python Coding Challenge** (Theme: GAME).

## How it works

1. Each round picks 5 different random games from a favourites list of 20 — one per question.
2. For each question, up to 5 clues about the secret game are revealed one at a time.
3. The player guesses after each clue.
4. Each question is marked as a percentage (100% for guessing on clue 1, down to 20% on clue 5, 0% if all 5 are missed).
5. The player is congratulated on each correct guess, and sees their overall round mark (average of all 5 questions) at the end.

## Requirements

- Python 3.7+

## Run it

```bash
python guess_the_game.py
```
