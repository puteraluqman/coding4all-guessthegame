# Teka Permainan

Satu permainan teka-teki Python yang dibina untuk **Cabaran Pengekodan Python "Coding for All"** (Tema: GAME).

## Cara ia berfungsi

1. Setiap pusingan memilih 5 permainan berbeza secara rawak daripada senarai 20 permainan kegemaran — satu bagi setiap soalan.
2. Bagi setiap soalan, sehingga 5 petunjuk tentang permainan rahsia didedahkan satu demi satu.
3. Pemain meneka selepas setiap petunjuk.
4. Setiap soalan dimarkahkan sebagai peratusan (100% jika teka pada petunjuk 1, menurun sehingga 20% pada petunjuk 5, 0% jika kelima-lima petunjuk gagal diteka).
5. Pemain diucapkan tahniah bagi setiap tekaan yang betul, dan melihat markah keseluruhan pusingan (purata kelima-lima soalan) pada akhirnya.

## Keperluan

- Python 3.7+

## Cara menjalankannya

```bash
python guess_the_game.py
```
